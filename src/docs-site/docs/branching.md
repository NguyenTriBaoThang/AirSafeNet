# Branching Strategy

See [BRANCHING.md](https://github.com/NguyenTriBaoThang/AirSafeNet/blob/main/BRANCHING.md) for the full guide.

## Quick Reference

```bash
# Start a feature
git checkout develop
git pull origin develop
git checkout -b feature/your-feature

# Commit
git commit -m "feat(frontend): add new widget"

# PR → develop → main
```
